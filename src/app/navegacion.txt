  X Mundos ( /admin/worlds )
    listado (editar / borrar) -> va a Escenarios

    X Escenarios del mundo X ( /admin/world/:id_world/scenarios )
      listado -> va a Editar Escenario
      añadir nuevo

    Editar Escenario ( /admin/world/:id_world/scenario/:id_scenario )
      editor del escenario
        indicar si es el escenario inicial y marcar punto de salida
        al pinchar en cada casilla poder modificar:
          fondo -> se elige de la lista
          objeto -> se elige de la lista pero se puede personalizar
          personaje -> se elige de la lista
